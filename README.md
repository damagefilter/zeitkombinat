## Zeitkombinat - The collectively owned time tracking tool

Zeitkombinat is a minimal, offline time tracking and invoicing tool built by a freelancer, for freelancers.  
It focuses on the only thing that matters: tracking your time and turning it into invoices.


Raw German efficiency. That is a nice way to say: there are no graphs and no fancy dashboards.
Tally estimated time against time spent. See your unbilled hours. Check open tasks.

It is boring.  
You are not supposed to spend more time in here than absolutely necessary.
Get out there and rake in some hours!


### Core Concepts

Zeitkombinat is built on a simple hierarchy:

- Projects contain stories
- Stories contain tasks
- Tasks contain work sessions

Each task can have an estimate and records the actual time spent.

Work sessions can be added manually or tracked using the stopwatch for tracking your time in ... realtime.  
These work sessions are then used to calculate invoice data (exported as CSV).


### Why Zeitkombinat?

Most time tracking tools try to be everything and the kitchen sink. And consequently demand delusional prices.
Zeitkombinat is intentionally small and boring:

- Tracks time
- Creates invoices
- Stays out of your way
